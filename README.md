# Minesweeper Game

A classic Minesweeper game with particle effects and rocket trail cursor.

## Play Now

[Play the game](你的Cloudflare Pages链接)

## Features

- 10x10 grid
- Particle effects
- Rocket trail cursor
- Responsive design
- Sound effects:
  - Click feedback
  - Cell reveal
  - Mine explosion
